# websocket-communication
# Overview
A simple WebSocket-based real-time communication system built using Node.js and the `ws` library.

# Features
- Real-time bidirectional communication
- Live event broadcasting
- Automatic reconnect handling
- Multiple client support

# Technologies
- Node.js
- WebSocket (ws)
- HTML
- JavaScript

# Run

Install dependencies:

```bash
npm install
```

Start the server:

```bash
node server.js
```

Open `client.html` in your browser.
